# CrewAI Adapter

CrewAI tool definitions and crew templates for Swarmix operations.

**Status:** Beta | **Minimum tier:** Team | **Requires:** [SDK Access](sdk-access.md)

## Features

- Pre-built CrewAI `BaseTool` wrappers for all Swarmix operations
- Dynamic tool manifest endpoint for auto-generating wrappers
- Crew templates for common multi-agent patterns
- Connection code templates with your org credentials pre-filled

## Requirements

- SDK Access plugin must be active (for API key authentication)
- Python 3.9+

## Getting Started

1. Enable the CrewAI Adapter plugin in the Swarmix dashboard
2. Go to the **Setup Guide** tab in the plugin settings
3. Download and run the setup script for your operating system
4. Download the crew template — a ready-to-run Python file

The setup script creates a virtual environment and installs everything automatically. No manual package installation or Python knowledge required.
